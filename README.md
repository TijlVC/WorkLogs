# Worklog Application

## Structuur
- `/frontend` - React applicatie
- `/backend` - Node.js API

## Setup
1. Clone de repository
2. Installeer dependencies:
   ```bash
   # Backend
   cd backend
   npm install

   # Frontend
   cd ../frontend
   npm install
   ```

3. Start de ontwikkelomgeving:
   ```bash
   # Backend
   cd backend
   npm run dev

   # Frontend
   cd ../frontend
   npm start
   ```

## Environment Variables
Maak een `.env` bestand aan in zowel de frontend als backend directory.
